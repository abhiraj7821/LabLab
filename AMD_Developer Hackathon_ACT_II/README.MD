# AMD Developer Hackathon: ACT II – AI Video Captioning Agent

## Project Overview

The **AI Video Captioning Agent** is a full-stack application that automatically generates engaging captions for videos using an AI-powered multi-agent workflow. Users can submit one or multiple video URLs, choose a preferred caption style (such as formal, sarcastic, tech humor, or non-tech humor), and receive optimized captions tailored to their selected tone.

The **frontend**, built with **React, Vite, and Tailwind CSS**, provides a modern neo-brutalist interface where users can enter video URLs, select caption preferences, track the AI processing pipeline in real time, and view or copy generated captions through an interactive dashboard.

The **backend** exposes REST APIs that accept video inputs, orchestrate the AI caption-generation pipeline, process multiple videos concurrently, and return structured caption results grouped by video. It manages the end-to-end workflow—from receiving user requests to generating context-aware captions using AI agents.

Together, the frontend and backend deliver a seamless AI-powered captioning platform that simplifies content creation, improves social media engagement, and demonstrates scalable multi-agent AI orchestration for real-world video applications.
